## 설명
정체불명 전염병의 탄생은 인류를 거의 멸종상태로 만들었다. 많은 사람이 피에 굶주린 좀비가 되었고, 이 재앙의 도시는 좀비들로 가득 차 있다. "가보자 지휘관"은 좀비로 가득찬 세상에서 군대의 지휘관이 되어 인류를 구하기 위해 좀비를 물리치는 게임이다.
